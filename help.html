Getting Started

Welcome to Hydro-M! Here's how to get started with the system.

1. System Overview

Hydro-M is a comprehensive hydroelectric power monitoring system that provides real-time data for the Ugandan cascade dams including Lake Nalubaale, Kiira HPP, Bujagali HPP, and Isimba HPP.

Real-time monitoring of water levels, dispatch, and discharge
Predictive analytics and water level forecasting
GIS monitoring and spatial data visualization
System alarms and alert management

2. First Steps

  1. Familiarize with the Dashboard: Start by exploring the main dashboard to understand the layout and available data.
  2. Check System Alarms: Review any active alarms in the right sidebar to understand current system status.
  3. Explore Dam Data: Use the left sidebar (toggle with menu button) to view detailed information for each dam.
  4. Download Data: Access the Download Data page to export monitoring data and reports.

System Features

Explore the powerful features and capabilities of the Hydro-M system.

Real-Time Monitoring

Live Data Display

Monitor real-time water levels, power dispatch, discharge rates, and environmental conditions across all cascade dams. Data updates automatically every 10 seconds.

Tips:
  • Use the carousel controls to switch between different dams
  • Hover over metric cards for detailed information
  • Check the live indicator to confirm data freshness

Water Level Prediction

AI-Powered Forecasting

Access predictive analytics for water level forecasting, including confidence levels, trend analysis, and time period predictions.

Tips:
  • Review confidence levels to assess prediction reliability
  • Monitor trend indicators for planning decisions
  • Use predictions for operational planning and risk assessment

GIS Monitoring

Spatial Data Visualization

Geographic Information System monitoring providing spatial context including coordinates, elevation, reservoir area, and satellite status.

Tips:
  • Use coordinates for precise location identification
  • Monitor satellite status for data reliability
  • Track reservoir area changes over time

System Alarms

Alert Management

Comprehensive alarm system with different severity levels (Critical, Warning, Info) and filtering capabilities by dam and category.

Tips:
  • Filter alarms by severity level for focused attention
  • Use dam-specific filters to isolate issues
  • Check recently resolved alarms for pattern analysis

Data Export & Reports

Comprehensive Data Access

Download raw monitoring data and access pre-generated reports (Weekly, Monthly, Quarterly, Annual) in multiple formats including CSV, Excel, JSON, and PDF.

Tips:
  • Use the modern select dropdowns for efficient data selection
  • Preview reports before downloading
  • Check recent downloads for quick re-access

Troubleshooting

Common issues and their solutions to help you resolve problems quickly.

Common Issues & Solutions

Data Not Updating

Problem: Real-time data appears stale or not updating.

Solution: Check your internet connection and refresh the page. Data updates every 10 seconds automatically.

Sidebar Not Opening

Problem: Left sidebar (dam information) doesn't open when clicking the menu button.

Solution: Ensure you're on a large screen (desktop view). On mobile devices, the sidebar is integrated into the mobile navigation.

Download Not Working

Problem: Unable to download data or reports.

Solution: Check that you've selected at least one dam and data type. Ensure your browser allows downloads and check for any popup blockers.

Alarms Not Displaying

Problem: System alarms are not showing in the right sidebar.

Solution: Navigate to the System Alarms page to view all alarms. The sidebar shows only recent alarms for quick access.

Mobile Layout Issues

Problem: Layout appears broken or difficult to use on mobile devices.

Solution: Use the mobile navigation menu (hamburger button) to access all features. The layout automatically adapts for mobile screens.

Performance Tips

Optimize Your Experience

  • Use a modern browser: Chrome, Firefox, Safari, or Edge for best performance
  • Enable JavaScript: Required for all interactive features
  • Check internet speed: Stable connection recommended for real-time data
  • Clear browser cache: If experiencing display issues
  • Use desktop view: For full feature access and better data visualization

Frequently Asked Questions

Quick answers to common questions about the Hydro-M system.

How often is the data updated?

Real-time data is updated every 10 seconds automatically. Historical data and reports are generated based on the selected time periods.

Can I access the system on mobile devices?

Yes! The system is fully responsive and optimized for mobile devices. Use the mobile navigation menu to access all features.

What data formats are available for download?

You can download data in CSV, Excel, JSON, and PDF formats. Reports are primarily available in PDF format.

How do I filter alarms by specific dams?

On the System Alarms page, use the dam filter dropdown to select specific dams and view only their associated alarms.

What do the different alarm severity levels mean?

Critical: Immediate attention required, Warning: Monitor closely, Info: Informational alerts.

How accurate are the water level predictions?

Predictions include confidence levels to indicate reliability. Higher confidence levels indicate more reliable predictions.

Can I export data for specific time periods?

Yes, you can select custom date ranges when downloading raw data. Reports are available for standard periods (weekly, monthly, quarterly, annual).

How do I contact support if I need help?

Use the Contact Us page to reach the Hydro-M support team. You can also refer to this Help section for common issues and solutions.
